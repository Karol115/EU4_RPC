Discord RPC for eu4. Works for all versions of eu4(steam, epic games etc).

This is separate app so you need to launch it when launching eu4 or create .bat that launches these apps(example below).


.bat example:
```
@echo off
start "" "{eu4_path}"
start "" "{eu4-RPC_path"
```

Then create shortcut to this .bat and change icon(Original exe shortcut->Change Icon->copy path and paste to shortcut for .bat).
Done!!!
